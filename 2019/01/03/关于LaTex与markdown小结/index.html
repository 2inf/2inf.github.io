¶Markdown

¶公式

插入公式与代码分别是用$与 ` 符号(～键)包裹所写部分

  • 在需要高亮的代码块的前一行及后一行使用三个反引号。
  • 同时第一行反引号后面，写代码块所使用的语言。
  • 使用 \left和 \right来创建自动匹配高度的 ()，[] 和 {} 。
    在每个公式末尾前使用\tag{行标}来实现行标。
